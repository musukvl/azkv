# Azure Key Vault Manager v1.0.3

## Refactoring and Reliability Release

This release focuses on internal architecture improvements and UI reliability, while preserving the end-user workflow introduced in v1.0.2.

## What's New

### ViewModel Architecture

- Introduced dedicated ViewModels for Key Vaults, Secrets, Versions, and Secret Details.
- Added a top-level `MainWindowViewModel` to coordinate screen state.
- Improved separation between UI rendering and business logic for easier maintenance and future feature work.

### Service Layer Improvements

- Added `IAzureKeyVaultDataService` abstraction for Key Vault data operations.
- Added `SubscriptionService` for subscription-related behavior.
- Added `SecretFormValidator` for centralized form validation.
- Added `SecretDateService` for date parsing and formatting consistency.
- Added `OperationResult` model for clearer operation outcomes.

### Secret Metadata and UX Updates

- Extended `Secret` model to include expiration date support.
- Updated UI components to better handle secret expiration metadata.
- Improved content type and version value formatting to display clean empty values instead of placeholders.

### Startup and Build Quality

- Refactored app initialization and error handling to use `IApplication` APIs.
- Enabled treating compiler warnings as errors to improve code quality and release confidence.

## Installation

### Homebrew (macOS/Linux)

```bash
brew tap musukvl/azkv
brew upgrade azkv
```

### Direct Download

Download the appropriate binary for your platform:

- **macOS (Apple Silicon)**: [azkv-1.0.3-osx-arm64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.0.3/azkv-1.0.3-osx-arm64.tar.gz)
- **macOS (Intel)**: [azkv-1.0.3-osx-x64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.0.3/azkv-1.0.3-osx-x64.tar.gz)
- **Linux (x64)**: [azkv-1.0.3-linux-x64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.0.3/azkv-1.0.3-linux-x64.tar.gz)
- **Windows (x64)**: [azkv-1.0.3-win-x64.zip](https://github.com/musukvl/azkv/releases/download/v1.0.3/azkv-1.0.3-win-x64.zip)

### Extract and Run

**macOS/Linux:**
```bash
tar -xzf azkv-1.0.3-*.tar.gz
chmod +x azkv
./azkv
```

**Windows:**
```powershell
Expand-Archive azkv-1.0.3-win-x64.zip
.\azkv\azkv.exe
```
