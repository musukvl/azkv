# Azure Key Vault Manager v1.1.0

## Major UX Refactor Release

This release delivers a large UI architecture refactor, improved interaction patterns, and tooling updates for both users and maintainers.

## What's New

### Modular UI Architecture

- Refactored the main TUI layout into dedicated panel components:
  - `KeyVaultsPanel`
  - `SecretsPanel`
  - `VersionsPanel`
  - `SecretDetailsPanel`
- Introduced dedicated dialog components for secret/version creation:
  - `AddSecretDialog`
  - `AddVersionDialog`
- Improved UI responsiveness by standardizing `StateChanged`-driven updates across view models and panels.

### Secrets and Versions UX Improvements

- Added **Reload** functionality in the Secrets panel with explicit user feedback in status updates.
- Improved layout and accessibility behavior across panels and list/table interactions.
- Kept secret metadata presentation (expiration/content type/value) consistent in the new panelized structure.

### Copy and Keyboard Behavior

- Updated secret value copy behavior so **Ctrl+C** copies the full secret value from the details pane.
- Added **Alt+C** as an additional copy shortcut.
- Updated help/about text to reflect current key bindings.

### CLI and Tooling Enhancements

- Switched argument parsing to `Spectre.Console.Cli` for cleaner command handling.
- Help output now initializes theme support and lists available themes.
- Added support files and project metadata for publishing `azkv` as a .NET global tool.
- Added `.NET Tool` installation instructions to `README.md`.

### Security and Maintainability Improvements

- Hardened Azure CLI invocation by using structured process argument passing (`ArgumentList`) in services.
- Improved command execution and error-handling flow in `AzureCliService` and `SubscriptionService`.

### Dependency Updates

- Updated `Terminal.Gui` to `2.0.0-develop.5125`.

## Installation

### Homebrew (macOS/Linux)

```bash
brew tap musukvl/azkv
brew upgrade azkv
```

### .NET Tool

```bash
dotnet tool install -g azkv
```

### Direct Download

Download the appropriate binary for your platform:

- **macOS (Apple Silicon)**: [azkv-1.1.0-osx-arm64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.1.0/azkv-1.1.0-osx-arm64.tar.gz)
- **macOS (Intel)**: [azkv-1.1.0-osx-x64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.1.0/azkv-1.1.0-osx-x64.tar.gz)
- **Linux (x64)**: [azkv-1.1.0-linux-x64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.1.0/azkv-1.1.0-linux-x64.tar.gz)
- **Windows (x64)**: [azkv-1.1.0-win-x64.zip](https://github.com/musukvl/azkv/releases/download/v1.1.0/azkv-1.1.0-win-x64.zip)

### Extract and Run

**macOS/Linux:**
```bash
tar -xzf azkv-1.1.0-*.tar.gz
chmod +x azkv
./azkv
```

**Windows:**
```powershell
Expand-Archive azkv-1.1.0-win-x64.zip
.\azkv\azkv.exe
```
