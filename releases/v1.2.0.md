# Azure Key Vault Manager v1.2.0

## Status UX and Error-Handling Release

This release improves operation feedback and error handling in the TUI by moving async workflow reporting into a clearer status-bar lifecycle.

## What's New

### Status Bar Lifecycle Improvements

- Added a loading spinner animation in the status bar for async operations.
- Standardized operation lifecycle messages across the app:
  - loading,
  - loaded,
  - error,
  - canceled (stale/superseded loads).
- Updated panel messaging to provide more explicit context (Key Vault, secret, and version operation targets).

### Error-Handling UX

- Removed intrusive error pop-up dialogs for common async load/refresh failures in panels.
- Errors are now surfaced through the status bar, with details retained for later inspection.
- Added clickable status behavior to open detailed operation error information on demand.

### Panel and Interaction Refinements

- Improved feedback consistency in `KeyVaultsPanel`, `SecretsPanel`, `VersionsPanel`, and `SecretDetailsPanel`.
- Kept stale background results from presenting as active outcomes by reporting canceled status instead.

### Project and Documentation Updates

- Added `LICENSE` file with MIT license text.
- Updated requirements to reflect the status-bar lifecycle model and canceled-state reporting behavior.

## Installation

### Homebrew (macOS/Linux)

```bash
brew tap musukvl/azkv
brew upgrade azkv
```

### .NET Tool

```bash
dotnet tool install -g azkv
```

### Direct Download

Download the appropriate binary for your platform:

- **macOS (Apple Silicon)**: [azkv-1.2.0-osx-arm64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.2.0/azkv-1.2.0-osx-arm64.tar.gz)
- **macOS (Intel)**: [azkv-1.2.0-osx-x64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.2.0/azkv-1.2.0-osx-x64.tar.gz)
- **Linux (x64)**: [azkv-1.2.0-linux-x64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.2.0/azkv-1.2.0-linux-x64.tar.gz)
- **Windows (x64)**: [azkv-1.2.0-win-x64.zip](https://github.com/musukvl/azkv/releases/download/v1.2.0/azkv-1.2.0-win-x64.zip)

### Extract and Run

**macOS/Linux:**
```bash
tar -xzf azkv-1.2.0-*.tar.gz
chmod +x azkv
./azkv
```

**Windows:**
```powershell
Expand-Archive azkv-1.2.0-win-x64.zip
.\azkv\azkv.exe
```
