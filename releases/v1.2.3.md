# Azure Key Vault Manager v1.2.3

## Terminal.Gui 2.1.0 Upgrade

This release upgrades Terminal.Gui to the stable `2.1.0` release and includes refactoring to align with its updated API.

## What's New

### Dependency Upgrades

- Updated `Terminal.Gui` from `2.0.0-rc.4` to `2.1.0` — the first stable release of Terminal.Gui v2, bringing improved keyboard handling, stable typed `ListView<T>`, and numerous bug fixes over the release-candidate series.

### Bug Fixes & Refactoring

- Refactored `AddSecretDialog` and `AddVersionDialog` to return typed results, improving reliability of dialog interactions.
- Fixed selection event handling in `SecretsPanel` and `VersionsPanel` to work correctly with Terminal.Gui 2.1.0's updated selection API.
- Improved command handling in `MainWindow` for more robust keyboard navigation.

## Installation

### Homebrew (macOS/Linux)

```bash
brew tap musukvl/azkv
brew upgrade azkv
```

### .NET Tool

```bash
dotnet tool install -g azkv
```

### Direct Download

Download the appropriate binary for your platform:

- **macOS (Apple Silicon)**: [azkv-1.2.3-osx-arm64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.2.3/azkv-1.2.3-osx-arm64.tar.gz)
- **macOS (Intel)**: [azkv-1.2.3-osx-x64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.2.3/azkv-1.2.3-osx-x64.tar.gz)
- **Linux (x64)**: [azkv-1.2.3-linux-x64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.2.3/azkv-1.2.3-linux-x64.tar.gz)
- **Windows (x64)**: [azkv-1.2.3-win-x64.zip](https://github.com/musukvl/azkv/releases/download/v1.2.3/azkv-1.2.3-win-x64.zip)

### Extract and Run

**macOS/Linux:**
```bash
tar -xzf azkv-1.2.3-*.tar.gz
chmod +x azkv
./azkv
```

**Windows:**
```powershell
Expand-Archive azkv-1.2.3-win-x64.zip
.\azkv\azkv.exe
```

## Prerequisites
