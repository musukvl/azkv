# Azure Key Vault Manager v1.2.2

## Dependency Refresh Release

This release upgrades all NuGet dependencies to their latest stable (or latest pre-release for Terminal.Gui) versions.

## What's New

### Dependency Upgrades

- Updated `Terminal.Gui` from `2.0.0-beta.92` to `2.0.0-rc.4` — the first release-candidate build, bringing Kitty Keyboard Protocol support, a new `MarkdownView`, `DropDownList`, typed `ListView<T>`, inline-mode apps, OSC terminal title, and numerous bug fixes.
- Updated `Spectre.Console` from `0.54.0` to `0.55.2`.
- Updated `Spectre.Console.Cli` from `0.53.1` to `0.55.0`.

## Installation

### Homebrew (macOS/Linux)

```bash
brew tap musukvl/azkv
brew upgrade azkv
```

### .NET Tool

```bash
dotnet tool install -g azkv
```

### Direct Download

Download the appropriate binary for your platform:

- **macOS (Apple Silicon)**: [azkv-1.2.2-osx-arm64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.2.2/azkv-1.2.2-osx-arm64.tar.gz)
- **macOS (Intel)**: [azkv-1.2.2-osx-x64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.2.2/azkv-1.2.2-osx-x64.tar.gz)
- **Linux (x64)**: [azkv-1.2.2-linux-x64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.2.2/azkv-1.2.2-linux-x64.tar.gz)
- **Windows (x64)**: [azkv-1.2.2-win-x64.zip](https://github.com/musukvl/azkv/releases/download/v1.2.2/azkv-1.2.2-win-x64.zip)

### Extract and Run

**macOS/Linux:**
```bash
tar -xzf azkv-1.2.2-*.tar.gz
chmod +x azkv
./azkv
```

**Windows:**
```powershell
Expand-Archive azkv-1.2.2-win-x64.zip
.\azkv\azkv.exe
```

## Prerequisites

- Azure CLI installed and configured
- Authenticated Azure session (`az login`)

## Verification

```bash
azkv --version
```

Expected output: `1.2.2`
