# Azure Key Vault Manager v1.2.1

## Dependency Refresh Release

This release updates the Terminal.Gui dependency to beta.92 to keep the TUI aligned with the current upstream toolkit.

## What's New

### Terminal.Gui Update

- Updated `Terminal.Gui` to `2.0.0-beta.92`.
- Keeps the application aligned with the current Terminal.Gui beta line used by the project.
- Reduces drift against upstream Terminal.Gui package changes for future maintenance and fixes.

## Installation

### Homebrew (macOS/Linux)

```bash
brew tap musukvl/azkv
brew upgrade azkv
```

### .NET Tool

```bash
dotnet tool install -g azkv
```

### Direct Download

Download the appropriate binary for your platform:

- **macOS (Apple Silicon)**: [azkv-1.2.1-osx-arm64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.2.1/azkv-1.2.1-osx-arm64.tar.gz)
- **macOS (Intel)**: [azkv-1.2.1-osx-x64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.2.1/azkv-1.2.1-osx-x64.tar.gz)
- **Linux (x64)**: [azkv-1.2.1-linux-x64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.2.1/azkv-1.2.1-linux-x64.tar.gz)
- **Windows (x64)**: [azkv-1.2.1-win-x64.zip](https://github.com/musukvl/azkv/releases/download/v1.2.1/azkv-1.2.1-win-x64.zip)

### Extract and Run

**macOS/Linux:**
```bash
tar -xzf azkv-1.2.1-*.tar.gz
chmod +x azkv
./azkv
```

**Windows:**
```powershell
Expand-Archive azkv-1.2.1-win-x64.zip
.\azkv\azkv.exe
```

## Prerequisites

- Azure CLI installed and configured
- Authenticated Azure session (`az login`)
