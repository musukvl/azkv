# Azure Key Vault Manager v1.2.4

## Terminal.Gui 2.4.7 Upgrade

This release upgrades Terminal.Gui to the latest stable version `2.4.7`, bringing performance improvements, bug fixes, and enhanced compatibility.

## What's New

### Dependency Upgrades

- Updated `Terminal.Gui` from `2.1.0` to `2.4.7` — the latest stable release with improved stability, performance optimizations, and numerous bug fixes over previous 2.x releases.

### Bug Fixes & Refactoring

- Added pragma directives to suppress obsolete warnings for `TextView` (marked as deprecated in favor of EditorView) — maintaining compatibility while using the existing proven implementation.
- Improved compatibility with Terminal.Gui 2.4.7's refined API while maintaining backward compatibility.

## Installation

### Homebrew (macOS/Linux)

```bash
brew tap musukvl/azkv
brew upgrade azkv
```

### .NET Tool

```bash
dotnet tool install -g azkv
```

### Direct Download

Download the appropriate binary for your platform:

- **macOS (Apple Silicon)**: [azkv-1.2.4-osx-arm64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.2.4/azkv-1.2.4-osx-arm64.tar.gz)
- **macOS (Intel)**: [azkv-1.2.4-osx-x64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.2.4/azkv-1.2.4-osx-x64.tar.gz)
- **Linux (x64)**: [azkv-1.2.4-linux-x64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.2.4/azkv-1.2.4-linux-x64.tar.gz)
- **Windows (x64)**: [azkv-1.2.4-win-x64.zip](https://github.com/musukvl/azkv/releases/download/v1.2.4/azkv-1.2.4-win-x64.zip)

### Extract and Run

**macOS/Linux:**
```bash
tar -xzf azkv-1.2.4-*.tar.gz
chmod +x azkv
./azkv
```

**Windows:**
```powershell
Expand-Archive azkv-1.2.4-win-x64.zip
cd azkv-1.2.4-win-x64
.\azkv.exe
```
