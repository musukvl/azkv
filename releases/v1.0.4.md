# Azure Key Vault Manager v1.0.4

## UX and Theming Release

This release adds runtime theme switching, introduces a theme command-line option, and improves reliability of Azure CLI error handling and dialog layouts.

## What's New

### Theme Support

- Added `-t` / `--theme` startup option to launch with a selected Terminal.Gui theme.
- Added a new **Theme** menu in the app for runtime theme switching.
- Added centralized theme initialization and apply logic via `ThemeProvider`.

### Better Error Reporting for Azure CLI

- Added `AzureCliException` to represent Azure CLI execution and parsing failures.
- Updated Azure CLI service methods to throw meaningful exceptions instead of silently returning empty values.
- Preserved Azure CLI stderr details in exception messages to improve troubleshooting (for example RBAC and authorization failures).

### Dialog Layout Improvements

- Improved **Add Secret** and **Add Version** dialogs to use dynamic sizing (`Dim.Auto`) and relative positioning.
- Reduced layout clipping risk across different terminal sizes and font metrics.

### Developer Workflow

- Added workspace release prompt command at `.github/prompts/release.prompt.md` to streamline future release operations.

## Installation

### Homebrew (macOS/Linux)

```bash
brew tap musukvl/azkv
brew upgrade azkv
```

### Direct Download

Download the appropriate binary for your platform:

- **macOS (Apple Silicon)**: [azkv-1.0.4-osx-arm64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.0.4/azkv-1.0.4-osx-arm64.tar.gz)
- **macOS (Intel)**: [azkv-1.0.4-osx-x64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.0.4/azkv-1.0.4-osx-x64.tar.gz)
- **Linux (x64)**: [azkv-1.0.4-linux-x64.tar.gz](https://github.com/musukvl/azkv/releases/download/v1.0.4/azkv-1.0.4-linux-x64.tar.gz)
- **Windows (x64)**: [azkv-1.0.4-win-x64.zip](https://github.com/musukvl/azkv/releases/download/v1.0.4/azkv-1.0.4-win-x64.zip)

### Extract and Run

**macOS/Linux:**
```bash
tar -xzf azkv-1.0.4-*.tar.gz
chmod +x azkv
./azkv
```

**Windows:**
```powershell
Expand-Archive azkv-1.0.4-win-x64.zip
.\azkv\azkv.exe
```
